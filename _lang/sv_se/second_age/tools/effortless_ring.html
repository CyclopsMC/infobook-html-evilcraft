Mödolös ring

Jag lyckades bygga en speciell ring som kan tillämpa löftet om produktivitet och hastighet vilket låter dig gå snabbare, hoppa högre och kliva upp på block lättare. Lägg detta föremål någonstans i din utrustning för att få dess effekter. Observera att du kan undvika att kliva upp på block genom att hålla ned Shift.
