Undead Tree

I was able to reanimate dead saplings, the trees that grow from this process are remarkable to say the least. They seem to give off a lot of Blood, but the saplings that drop still seem to be dead somehow. Combining the wood with a Dark Gem will result in some sticks which will be useful some day.

Killing a sapling is as simple as shearing a living one in a crafting grid. Is that tree growing in a desert?

Dropping sticks made from this wood always seems to rotate in a certain direction. Could it be pointing to somewhere?
