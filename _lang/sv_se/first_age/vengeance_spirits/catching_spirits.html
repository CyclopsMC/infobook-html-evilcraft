Hur man fångar andar

På en del av mina äventyr upptäckte jag några konstiga lila lådor som innehöll några distorterade andar. Det ser ut som att dessa andar fångades för länge sedan och har brutits ned sedan dess. Jag vet dock inte om man kan återställa dem till sin ursprungliga form.

Jag lyckades demontera dessa lådor för att själv kunna tillverka dem. Under tiden upptäckte jag också hur man fryser andar som dessa lådor kan absorbera. Jag förbättrade hämndringen med denna teknologi för att skapa ett hämndringsfokus som kan användas för att sikta på målandar och fånga dem. Jag kanske kan använda dessa fångade andar för någonting senare...
